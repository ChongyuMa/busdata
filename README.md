# busdata

## 数据描述
本数据集包含公交车辆运行数据，用于前向碰撞风险预警研究。

### 数据采集地点
青岛市部分公交线路（1路至10路）。

### 采集方法
通过车载GPS和雷达传感器采集，时间分辨率为10秒，包含车辆速度、相对速度、与前车距离、GPS坐标、加速度、制动状态等字段。

### 数据格式
JSON 格式，每条记录包含如下字段：
- 当前车辆速度 (km/h)
- 相对速度 (km/h)
- 与前车距离 (m)
- 车辆编号 (如 "1路")
- 时间戳 (ISO 8601)
- GPS经度、纬度
- 加速度 (m/s²)
- 制动状态 (true/false)
